🔑 Key Concepts
  • [01]
    Fieldbus vs Industrial EthernetFieldbus uses dedicated cables. Industrial Ethernet rides Cat5e/Cat6 — higher bandwidth, easier integration.
  • [02]
    Hard vs Soft Real-TimeHard RT (PROFINET IRT, EtherCAT) guarantees <1ms deterministic cycles. Soft RT tolerates jitter — fine for I/O, not motion.
  • [03]
    OT / IT ConvergenceOPC-UA bridges PLCs/SCADA with ERP/cloud. Vendor-neutral with semantic data modeling.
  • [04]
    Modbus Function CodesFC01 Read Coils · FC03 Read Holding Regs · FC06 Write Single Reg · FC16 Write Multiple Regs
  • [05]
    Protocol Selection RuleLegacy → Modbus · Factory → PROFIBUS/NET · NA plants → EtherNet/IP · SCADA → DNP3 · Cloud → OPC-UA
